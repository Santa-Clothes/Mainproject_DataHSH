## 🔍 재현성 분석

### 변동계수 (Coefficient of Variation)

```
CV = (표준편차 / 평균) × 100%

Top-1 CV: 2.56% → 매우 안정적 ✅
Top-5 CV: 1.31% → 매우 안정적 ✅
MRR CV:   1.02% → 매우 안정적 ✅
```

### 평가 기준
- **CV < 5%**: 매우 안정적 ✅
- **CV < 10%**: 안정적
- **CV ≥ 10%**: 변동성 있음

### 결론
**모든 지표가 CV < 5% → v5는 매우 안정적이고 재현 가능** ✅
